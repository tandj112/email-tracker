Setup Guide

1
Go to console.cloud.google.com, create a project.
2
APIs & Services → Library, enable Gmail API.
3
OAuth consent screen, select External, add your email as test user.
4
Credentials → Create → OAuth Client ID → Web. Add http://localhost:8080 as authorized origin.
5
Copy Client ID. Get Anthropic key from console.anthropic.com.

Cross-Device Sync (Optional)

6
Go to github.com/settings/tokens, create a token with gist scope only.
7
Paste the token above. Your tasks will sync across all devices where you use this token.
